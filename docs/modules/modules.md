# POS Modules

The POS core should stay small. Extra capabilities should live as optional modules.

## Core

- Today
- Capture
- Identity
- Habits
- Planning
- Settings

## Optional Modules

- Learning
- Research
- Health profile
- Women's health
- Life clock
- Finance
- Environment and integrity
- Career
- Immigration
- Language learning
- Engineering projects
- Accountability

Each module should answer one clear question and should be removable without breaking the core experience.
