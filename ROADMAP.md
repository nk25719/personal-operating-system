# Roadmap

- Rename product to POS
- Today-first interface
- Capture-first interaction model
- Optional modules layer
- Initial identity, habit, planning, health, environment, and AI services

- Apple Calendar integration refinement
- Apple Reminders integration
- Notion sync

## v0.4.0 - Knowledge & Research Engine

- Topic maps

- Paper and project linking
- Learning plan generation

- Purpose review
- Values tracking
- Moral reminders
- Environment review
- Accountability contacts
- Integrity trend over time

- Secure account/data layer
- Stable iOS build
- Private-by-default settings
- Production AI backend
- Production integrations
- User testing feedback incorporated
